# TO DO APP
Full STack MERN App 

## ** What you need to run this code **
1. Node 
2. NPM 
3. MongooseDb



## Run the project by one terminal
`npm run dev`
Runs both the client app and the server app in development mode. Open http://localhost:3000 to view the project in the browser.
